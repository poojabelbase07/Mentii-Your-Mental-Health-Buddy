# MENTII - YOUR MENTAL HEALTH BUDDY

Mentii is a web application focused on supporting mental well-being by helping users track moods, journal thoughts, and reflect on their emotional health.

## Features
- User authentication (login / signup)
- Mood tracking
- Personal journaling
- Mental health tips and resources
- Responsive and simple UI

## Getting Started

Clone the repository and run the project locally:

```bash
git clone https://github.com/poojabelbase07/Mentii-Your-Mental-Health-Buddy.git
cd Mentii-Your-Mental-Health-Buddy
npm install
npm start
